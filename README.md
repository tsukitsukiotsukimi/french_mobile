# スマホ演習版｜統合フランス語学習Web教材

## 開き方

1. ZIPを展開します。
2. `index.html` をスマホまたはPCのブラウザで開きます。
3. 最上部の「演習モードを始める」から、1問ずつ解けます。

## 特徴

- 問題はボタン回答または「解答を見る」ボタンで進行
- 回答直後に正解・解説画面へ切り替え
- 「次の問題へ」で解き進める形式
- 教科書・100会話・600問題の原文も掲載
- 検索UIなし
- オフライン利用可能

## 内容保持

元Markdown 3本は `source_markdown/` にそのまま同梱しています。SHA-256は `manifest.json` に記録しています。
