# スマホ集中版（検索UI削除） 統合フランス語学習Web教材

## 開き方

1. ZIPを展開する
2. `index.html` をスマホまたはPCのブラウザで開く
3. オフラインでも利用可能

## スマホ向け追加機能

- 章ごとの折りたたみ表示
- 下部固定ナビ（教科書 / 会話 / 問題 / 上へ）
- 前回読んだ位置への復帰
- 文字サイズ切替
- ランダム会話・ランダム問題SECTIONジャンプ
- - 長い表・会話コードのスマホ表示最適化
- `content-visibility` による描画負荷軽減

## 内容保持

本文HTMLに加えて、元Markdown 3ファイルを `source_markdown/` に同梱しています。内容保持確認用のSHA-256は `manifest.json` を参照してください。

## 変更点

- 教材内検索バーと下部ナビの検索ボタンを削除しました。
- 表示操作は「表示操作」パネルに集約し、本文学習の邪魔にならない構成にしました。
